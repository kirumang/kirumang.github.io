Research Landscape

Today's batch clusters most heavily around world models and structured intermediate representations as the connective tissue between vision-language reasoning and low-level robot action. Masked Visual Actions expresses control as a partially revealed pixel trajectory inside a video model itself, so the same checkpoint can act as a forward dynamics model or invert to synthesize robot motion from a desired outcome. RoboInter1.5 pushes the same idea to dataset scale (230k episodes, ten-plus annotation types) and treats those representations as a bidirectional interface regularizing both policies and simulators. WorldScape Policy 2.0 and Agentic Real2Sim extend the pattern to long-horizon memory and real-to-sim conversion respectively, while Cognitive Dual-Process Planning applies the same structured-reasoning instinct to autonomous driving. Collectively these papers suggest the field is converging on "reasoning scaffolding" — explicit intermediate structure — as the mechanism for getting foundation models to control robots reliably, rather than relying on end-to-end scale alone.

A second thread is a small but pointed cluster of formal-methods papers attacking Signal Temporal Logic from three different angles. STL-GCS embeds STL into trajectory planning via time-varying convex sets with forward-invariance guarantees; Correct-by-Construction BT Synthesis compiles STL directly into behavior trees, closing a real timing-expressiveness gap in the LTL-based synthesis literature; and On the Limits of Sampling-Based Reachability proves the opposite kind of result — a matching upper and lower bound showing that sampling-based reachability's exponential dependence on state dimension and horizon is intrinsic, not a limitation of any specific method. Seeing rigorous synthesis and rigorous impossibility results in the same day's batch is a useful reminder that formal robotics is maturing on both fronts simultaneously: what can be guaranteed, and where guarantees provably run out.

A third theme is aerial and ground navigation as a systems problem, framed explicitly by Computing on the Fly's call for infrastructure-scale drone coordination (12 named technical gaps, from edge-cloud continuum to "reliable fleets from non-deterministic agents"). The rest of the day's navigation papers read like partial answers to that call: GPS-denied localization (NGPS), real-time SDF mapping and planning onboard a quadrotor (OREN-Bubble*), hierarchical LLM control for network-aware UAV coordination (ITNTNs), and confidence-aware command issuance between UAVs and UGVs (Confidence-Gated Heading Alignment). A recurring sub-debate, visible in Beyond Transformers's finding that longer context doesn't help Transformer-based navigation policies, mirrors the world-model cluster's emphasis on structure over scale: several teams are independently concluding that how state is represented and updated matters more than how much of it is fed to a bigger model.

Masked Visual Actions for Unified World Modeling

2026-07-21 cs.CV, cs.RO Maneesh Agrawala (h=78)
Hadi Alzayer, Wenlong Huang, Haonan Chen, Christopher Luey, Lvmin Zhang
Core Contributions
  • Encodes actions as partially revealed pixel trajectories inside the video itself, rather than as a separate control-token stream — action and observation live in the same representation space the video model already learned.
  • The same checkpoint runs both directions: reveal a robot trajectory and it behaves as a forward dynamics model predicting the scene's response; reveal a desired object trajectory and it inverts to synthesize the robot motion that would cause it.
  • Remarkably data-efficient — only 15 hours of masked fine-tuning examples yield one model that generalizes across scenes and multiple robot embodiments, rather than requiring embodiment-specific fine-tuning.
  • Imagined rollouts correlate with real execution outcomes well enough to serve as a policy-evaluation and planning signal, ranking candidate futures without physically executing them.
  • Reframes the action-interface problem itself rather than bolting a new conditioning module onto a pretrained video backbone, closing the usual alignment gap between symbolic actions and visual priors.
Show abstract

Video models absorb rich priors over how the visual world moves, interacts, and responds to contact, making them promising substrates for robotic world modeling. The central challenge is how to communicate action to such models in a form aligned with the visual space in which they learned these interaction priors, yet still grounded in physical manipulation. We introduce Masked Visual Actions, a pixel-space control interface that expresses action as a partially revealed trajectory of an arbitrary entity in a video. Revealing robot motion makes the model act as a forward dynamics model that predicts the scene's response to low-level robot actions, while revealing desired object motion makes the same model recover robot behavior consistent with that outcome. Finetuned with only 15 hours of masked examples from real videos and simulation, a single checkpoint achieves strong visual fidelity and controllability across diverse scenes and multiple embodiments. In downstream manipulation settings, the model produces imagined rollouts whose outcomes correlate with real-world execution for policy evaluation, improves decision making by ranking candidate futures in model-based planning, and supports inverse modeling by synthesizing robot motion from desired object motion.

2026-07-21 cs.RO Yilun Chen (h=30)
Ziqin Wang, Hao Li, Weijun Wang, Junhao Cai, Jia Zeng
Core Contributions
  • Scales dense intermediate-representation annotation to 230k manipulation episodes across 571 scenes, each labeled with ten-plus representation types (affordances, contact points, grasp poses, motion traces) instead of sparse action-only labels.
  • Treats intermediate representations as a bidirectional interface — regularizing what low-level policies are allowed to do while simultaneously constraining what the world-model simulator is allowed to imagine — rather than as merely an interpretability layer.
  • Introduces RoboInter-VQA to directly benchmark whether a VLM's intermediate-representation reasoning actually improves, closing a gap where such representations were used but rarely evaluated in isolation.
  • Compares implicit, explicit, and modular plan-then-execute action paradigms on the same representation stack, isolating how much benefit comes from the representations themselves versus how they're consumed.
Show abstract

Existing robot datasets remain expensive to curate, embodiment-specific, and insufficiently annotated with the fine-grained structure required for generalizable reasoning, execution, or long-horizon environment dynamics simulation. Building on our prior work, RoboInter1.0, we present RoboInter1.5, an extended and holistic suite of intermediate representations for both robotic manipulation and embodied world modeling. RoboInter1.5 provides a unified resource of data, benchmarks, and models centered on dense manipulation-oriented intermediate representations. Specifically, RoboInter-Data contains over 230k manipulation episodes across 571 scenes with dense per-frame annotations covering more than ten types of intermediate representations, including subtasks, primitive skills, object and gripper grounding, segmentation, affordance, grasp poses, contact points, motion traces, etc. Built upon these annotations, RoboInter-VQA introduces spatial and temporal embodied VQA tasks to benchmark and improve the intermediate-representation reasoning capabilities of our RoboInter-VLM. To better model the physical world, we further introduce RoboInter-World, which leverages intermediate representations as structured conditioning signals for controllable prediction of future world states. Extensive evaluations demonstrate that RoboInter1.5 provides a unified spatiotemporal scaffolding for intermediate representations. Rather than treating intermediate representations merely as interpretable signals, RoboInter1.5 conceptualizes them as a bidirectional interface that both regularizes low-level action spaces and constrains the latent rollouts of open-world physical simulators.

Agentic Real2Sim: Physics-based World Modeling with Vision-Language Agents

2026-07-21 cs.RO, cs.AI Chenfanfu Jiang (h=16)
Guanxiong Chen, Qianjun Xia, Jiawei Peng, Heng Zhang, Bole Ma
Core Contributions
  • Uses vision-language agents to orchestrate the entire real2sim pipeline — geometry recovery, physical parameter inference, scene assembly — replacing manual tuning of visual foundation models and brittle pipeline glue with agentic decision-making.
  • Spans rigid-object, deformable-object, and humanoid-motion scenes within one framework, whereas real2sim pipelines are typically domain-specific and don't transfer across these categories.
  • Shows agentic decisions can be driven by an open-weight VLM at a small fraction of frontier-model cost while reaching comparable conversion success rate — a meaningful cost argument for scaling real2sim data generation.
  • Targets the resulting "episodic twins" specifically for downstream policy learning and evaluation, positioning real2sim conversion as training-data infrastructure rather than a one-off visualization tool.
Show abstract

Real-to-sim conversion for robotic interaction with objects remains labor-intensive because it requires more than visual reconstruction: a streamlined real2sim process must recover scene geometries and object states, infer physical parameters, and assemble actors, objects, cameras, poses, and trajectories into a runnable physical simulation. Today this process still depends on manual tuning of visual foundation models, mesh cleanup, coordinate-frame alignment, and brittle workflow glue across visual perception tools and simulators. We introduce \textit{Agentic Real2Sim}, a framework for generalized physical world modeling with vision-language agents, converting a real-world recording of object-robot interaction into a simulatable episodic twin which preserves observations, geometries, robot interactions, and object states. We evaluate Agentic Real2Sim on rigid-object manipulation, deformable-object interaction, and humanoid motion scenes, spanning domains that are usually handled by separate Real2Sim pipelines, marking a first step toward scalable conversion. The framework's agentic decisions can be driven by an open-weight VLM backend at a small fraction of the cost of frontier models, while attaining comparable conversion success rate. We aim to use the resulting real-world-aligned twins for downstream robotics tasks, specifically policy learning and evaluation. The project site is available at https://ericchen321.github.io/agentic_real2sim.github.io/.

WorldScape Policy 2.0: Empowering Steerable World Action Modeling with Reasoning-Augmented Memory

2026-07-21 cs.RO Chen Gao (h=7)
Haisheng Su, Zongdai Liu, Xin Jin, Haoxuan Dou, Chengming Hu
Core Contributions
  • Splits memory into causal short-term visual memory feeding DiT prefill and long short-term event memory organized into global-history, local-active, and event-boundary tiers, addressing the limited temporal context of prior World Action Models.
  • Retrieved history becomes an implicit subgoal condition for planning tokens, and "semantic forcing" pushes event-level instruction semantics into that latent pathway — moving past the coarse episode-level, text-only supervision of earlier WAMs.
  • Introduces ManipEvent-5M, a roughly 5-million-event-segment pretraining dataset with aligned actions, instructions, subtask captions, goal images, and video — a scale of event-grounded annotation not previously available for this model class.
  • Supports three separate control interfaces — high-level instructions, fine-grained text, and goal-image or video-context prompts — within one architecture rather than committing to a single conditioning modality.
Show abstract

World Action Models (WAMs) offer a promising paradigm for robotic manipulation by jointly modeling visual state transitions and robot actions. However, existing WAMs are constrained by limited temporal context, coarse episode-level language supervision, and predominantly text-only conditioning, which hinder task-progress tracking and fine-grained language-video-action grounding while limiting visual-context reasoning and cross-embodiment transfer. In this paper, we introduce WorldScape Policy 2.0, a controllable WAM with reasoning-augmented long short-term memory. Its causal short-term visual memory supplies recent observations as DiT prefill to preserve local interaction dynamics, while its long short-term event memory organizes historical VLM outputs into global-history, local-active, and event-boundary representations for progress-aware retrieval. The retrieved history augments perception and autoregressively generated planning tokens, yielding an implicit subgoal condition for autonomous planning; semantic forcing further transfers event-level instruction semantics into this latent planning pathway. These designs provide a unified interface for autonomous planning from high-level instructions and controllable execution from fine-grained text, goal-image, or video-context prompts. Experiments in both simulation and real-world platforms demonstrate superior capabilities in long-horizon autonomous planning, fine-grained instruction following and in-context adaptation.

STL-GCS: A Planner-Controller Framework for Signal Temporal Logic via Graphs of Time-varying Convex Sets

2026-07-21 eess.SY, cs.RO Dimos V. Dimarogonas (h=65)
Nicola De Carli, Gregorio Marchesini, Dimos Dimarogonas
Core Contributions
  • Departs from the typical "encode STL as MILP constraints" approach; time-varying convex sets are built so that forward invariance directly implies specification satisfaction with a guaranteed robustness margin, turning logic satisfaction into geometric containment.
  • Casts trajectory planning as a shortest-path search over Graphs of Convex Sets in a joint time-configuration space, letting B-splines guarantee STL satisfaction, collision avoidance, and smoothness simultaneously rather than checked post hoc.
  • Reuses the same time-varying sets from planning inside the tracking controller, so execution-time disturbances are handled under the same formal guarantees instead of a separately tuned low-level loop.
  • Validated on real space robotic hardware rather than simulation only — an uncommon bar for STL-based planners in this literature.
Show abstract

We present a unified trajectory planning and control framework for the satisfaction of Signal Temporal Logic (STL) specifications defined over convex predicates. At the planning layer, STL tasks are encoded as time-varying convex sets in configuration space, specifically designed so that forward invariance of the system with respect to these sets implies satisfaction of the specification with a prescribed robustness margin. This representation is then lifted to the joint time--configuration space and combined with the Graphs of Convex Sets (GCS) framework, yielding a shortest-path formulation of the planning problem over convex spatio-temporal sets. Trajectories are parameterized by B-splines, which enable continuous-time enforcement of STL satisfaction, collision avoidance, and smoothness constraints. At the control layer, the same time-varying sets used for planning are exploited to design a feedback controller that tracks the planned trajectory while prioritizing satisfaction of the STL specification during execution in the presence of tracking errors and model mismatch. We validate the proposed approach in simulation and in real-world experiments on space robotic platforms.

Stochastic Multi-Objective Kinodynamic Planning Against Adversaries

2026-07-21 cs.RO Dimitra Panagou (h=32)
Thomas Marshall Vielmetti, Daniel Cherenson, Dimitra Panagou
Core Contributions
  • Targets a specific failure mode of chance-constrained planners: evaluating risk over open-loop trajectories makes them overly conservative when adversaries or other agents react to the ego vehicle's own behavior.
  • Shifts planning from single trajectories to sequences of closed-loop policies, with risk estimated via Monte-Carlo particle rollouts embedded directly into tree construction rather than computed after the fact.
  • Introduces two algorithms — SMO-RRT (probabilistically complete) and SMO-SST (prunes for speed, trading away completeness) — giving an explicit choice between guarantees and runtime.
  • Derives finite-sample bounds on chance-constraint violation that hold under non-Gaussian, state-dependent uncertainty, extending beyond the Gaussian-noise assumptions common in this literature.
Show abstract

This paper addresses multi-objective kinodynamic planning in environments with stochastic hybrid adversaries that probabilistically transition to adversarial modes based on the ego state. The goal is to construct the Pareto-front of paths that trade off execution cost and the probability of safety constraint violation (risk). Existing chance-constrained planners evaluate risk over open-loop trajectories, yielding overly conservative solutions that fail to account for ego-agent reactivity. To address this limitation, we shift the planning space to sequences of closed-loop policies, and integrate sample-based risk evaluation directly into tree construction via Monte-Carlo particle rollouts. We first introduce Stochastic Multi-Objective RRT (SMO-RRT), for which we prove probabilistic completeness, followed by Stochastic Multi-Objective Stable Sparse RRT (SMO-SST), which leverages selective pruning to improve numerical performance at the cost of completeness. For both algorithms, we derive a finite-sample bound on the probability of chance constraint violation for systems with non-Gaussian, state-dependent uncertainty, enabling probabilistically safe planning in a broad class of environments applicable to multi-agent systems, social navigation, and autonomous driving.

On the Limits of Sampling-Based Reachability: Geometry, Dynamics, and Sample Complexity

2026-07-21 cs.RO Enrique Mallada (h=30)
Jixian Liu, Ihab Tabbara, Hussein Sibai, Enrique Mallada
Core Contributions
  • Reframes sampling-based reachability as geometric support estimation, formally connecting coverage-probability guarantees to Hausdorff-distance accuracy — a link prior work left implicit.
  • Proves accuracy is achievable once two regularity conditions hold (positive reach of the initial set's complement, Lipschitz dynamics), specifying exactly when sampling-based methods can be trusted rather than assumed universally applicable.
  • Delivers a rare complete complexity picture: an upper bound of Õ((e^3LT/r)^n) samples matched by a minimax lower bound of Ω((e^LT/r)^n), proving the exponential blow-up with dimension and horizon is intrinsic, not an artifact of any specific algorithm.
  • Shows empirically that adversarial sampling only improves constants, not the exponential scaling — tempering hope that smarter sampling alone can beat the curse of dimensionality in reachability analysis.
Show abstract

Reachability analysis is central to safety-critical control, robotics, and neural network verification, but classical computational methods, such as Hamilton--Jacobi reachability and set propagation, scale poorly with state dimension. Sampling-based methods have emerged as a promising alternative, often providing finite-sample guarantees that bound the probability-mass left uncovered. However, an explicit account of how the geometry of the initial set, the dynamics, and the sampling law affect the accuracy of the estimator is not fully available in the literature. We study this by casting sampling-based reachable-set recovery as geometric support estimation over a family of problems specified by an initial set, its dynamics, and a sampling law. First, we identify two regularity properties, positive reach of the initial set's complement and Lipschitz continuity of the dynamics, that together make recovery well-posed: a probability-mass coverage guarantee can be upgraded to accuracy $r$ in Hausdorff distance. Second, we bound the resulting sample complexity: recovery is achievable with $\tilde{\mathcal{O}}\big((e^{3LT}/r)^n\big)$ samples, exponential in both the state dimension and the time horizon. Third, we show that neither can be removed: an minimax lower bound of $Ω\big((e^{LT}/r)^n\big)$ holds for every estimator, so the exponential dependence on dimension and the degradation over the horizon are both intrinsic, not artifacts of a particular method. Experiments on nonlinear systems confirm that adversarial sampling improves constants but not the scaling.

Pose-Parameterized Motion Planning and CBF-QP Self-Collision Filtering for a Long-Reach Drilling Boom

2026-07-21 cs.RO, eess.SY J. Mattila (h=23)
Mehdi Heydari Shahna, Tuomo Kivelä, Jouni Mattila
Core Contributions
  • Targets a concrete industrial autonomy gap: moving a real long-reach drilling boom (Sandvik SB60) from operator-supervised control toward autonomous, self-collision-free pose sequencing.
  • Combines offline detour planning (screening waypoints against 23 rod-to-body distance constraints) with an online capsule-based CBF-QP filtering joint velocities against 14 capsule-pair constraints, splitting the collision-avoidance burden between planning time and execution time.
  • Uses one fixed parameter set across different drilling tasks with no target-specific retuning — a practical deployment requirement often glossed over in academic motion-planning work.
  • Validated across several hundred thousand samples in a manufacturer-built Simscape Multibody model with zero IK failures and millimetre-level final-position error.
Show abstract

Long-reach drilling booms must reach successive poses without self-collision. Moving from operator-supervised control toward autonomy requires collision-aware motion planning and execution. For the Sandvik SB60, this study adapts established methods by integrating pose-parameterized planning with a capsule-based control barrier function quadratic program (CBF-QP) in measured-state inverse kinematics (IK). A fixed task-specific parameter set within each task generates waypoints, detours, timed references, and chained motion without target-specific retuning. The offline detour planner screens candidate waypoints using 23 selected rod-segment-to-body-region distances, whereas the online CBF-QP filters joint velocities using 14 configured capsule-pair constraints from a nine-primitive whole-body capsule model. Correct-by-Construction Behavior Tree Synthesis from Signal Temporal Logic Specifications with Application to Robotic Missions

2026-07-21 cs.RO Liang Han (h=4)
Jiaheng Dong, Jingyi Huang, Liang Han
Core Contributions
  • Extends correct-by-construction behavior tree synthesis from LTL, which can't express timing, to full Signal Temporal Logic, closing a real capability gap for BTs used in timed robotic missions.
  • Abstracts the workspace into a zone graph and augments the state space to jointly track logical progress and timing constraints, rather than handling timing as a separate post-hoc check.
  • A hierarchical fixed-point algorithm computes winning sets for a broad STL fragment (safety, reachability, response, recurrence, persistence) and directly yields BT subtrees with runtime constraint functions — the synthesis output is already the deployable structure, not an intermediate automaton needing translation.
  • Validated on a physical quadrotor across six STL specifications with proven strictly positive robustness, not just simulated satisfaction.
Show abstract

Behavior Trees (BTs) are widely adopted for complex task execution in robotics, providing modular, reactive control but lacking formal guarantees. However, existing correct-by-construction synthesis from Linear Temporal Logic (LTL) cannot express quantitative timing constraints. This letter synthesizes correct-by-construction BTs from Signal Temporal Logic (STL) specifications. The workspace is modeled as a timed transition system and abstracted into a zone graph, and an augmented state space tracking both logical progress and timing constraints is introduced. A hierarchical fixed-point algorithm computes winning sets for an STL fragment encompassing safety, reachability, response, recurrence, and persistence, yielding BT subtrees with a runtime constraint function. Correctness guarantees are proven and complexity bounds are derived. Simulations demonstrate specification satisfaction with strictly positive robustness, and a physical quadrotor experiment with six STL specifications validates practical deployability.

From Distances to Trajectories: Real-Time Signed Distance Function Mapping and Distance-Accelerated Motion Planning for UAVs

2026-07-21 cs.RO, cs.AI, cs.CV, eess.SY La Jolla (h=16)
Jason Stanley, Zhirui Dai, Qihao Qian, Tzu-Chin Ho, Tianxing Fan
Core Contributions
  • Argues mapping and planning should share one representation — a signed distance function — rather than separate stages built on binary occupancy, since SDFs carry graded distance-to-obstacle information planners can exploit directly.
  • OREN pairs an explicit octree prior with an implicit neural residual, combining the speed of volumetric methods with the accuracy and differentiability of neural SDF reconstruction, reporting 22% better SDF estimation than baselines.

Show abstract

Autonomous flight in cluttered environments requires a robot to build a geometric map of its surroundings and plan safe, dynamically feasible trajectories, all onboard and in real time.
